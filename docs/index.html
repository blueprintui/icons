Getting Started

BlueprintUI Icons provides a flexible, extensible, and easy-to-use set of icons for use in any web UI.

400+ Open Source Svg Icons from Clarity Design Lightweight and Fully Tree-Shakable Web Components that work in any framework 
import '@blueprintui/icons/include.js';
import '@blueprintui/icons/shapes/user.js';

Customize

Provide your own custom SVG icons easily. Register icons to the icon service to add aditional icons to the element.

import { IconService } from '@blueprintui/icons';

IconService.add({
  name: 'logo',
  type: {
    default: '...svg...',
    solid: '...svg...'
  }
});
<bp-icon shape="logo"></bp-icon>
